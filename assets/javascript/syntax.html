Comparators and Equality

=== Is equal to same data type

== Is equal to same value

!== Is not equal to

> Is greater than

< Is less than

>= Is greater or equal to

<= Is less or equal to

Combining Comparators

&& isequal and

|| Is equal or

! Is equal not

For Example

if (condition && condition) {

Run this block of code

} else if (condition || condition) {

Run this block of code

} else (condition) {

Run this block of code

}

Arrays

To create an array create a variable, give it a name = add collection of data in [ ] separating each data by ,(coma)

var names = ["Angela", "John", "Josh", "James"]

To console.log an array

console.log(name); OR console.log(name[0]);
